# Productivity Manager
### Vasiescu Adriana

## Descriere
Această aplicație reprezintă o agendă interactivă - o unealtă cu ajutorul căreia utilizatorul își poate simplifica managementul activităților pe care le are de indeplinit pe parcursul unei perioade îndelungate.

## Obiective
Interfață intuitivă 
* aspect minimalist, aerat și ușor de înțeles;
* Categorii editabile.
* Tabele cu orare pentru vizualizare structurată

## Functionalitati/Exemple utilizare
In aceasta aplicatie , utilizatorul poate introduce task-urile pe care le are de indeplinit si data scadenta. Aceste task-uri vor putea fi organizate in functie de data, prioritate sau etichetata intr-un mod care ajuta filtrarea si sortarea taskurilor pentru o vizualizare mai usoara (orar studii, curățenie, rutine, etc.)

Aplicația va genera notificări de incurajare atunci cand un task se va apropia de dead-line.

Java, JavaFX, SQLite,JDBC
